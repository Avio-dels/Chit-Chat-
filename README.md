# 💬 ChatApp

A **real-time chat application** built with **PHP, MySQL, HTML, CSS & JavaScript**, ready to run locally using **XAMPP**.  
Communicate instantly with a simple, user-friendly interface! 🚀

---

## ✨ Features

- 💬 Real-time messaging  
- 👥 Multi-user support  
- 🖥️ Responsive & user-friendly interface  
- 📂 Optional file upload (if implemented)  
- ⚡ Easy local setup with XAMPP  

---

## 🛠 Technologies Used

- **Frontend:** CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** Apache (via XAMPP)  

---

## ⚡ Quick Setup Guide

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/Avio-dels/Chat-App.git
````

### 2️⃣ Move to XAMPP

* Copy `Chat-App` folder to `C:\xampp\htdocs\`
* Optionally rename it to `chatapp`

### 3️⃣ Start XAMPP Services

* Open **XAMPP Control Panel**
* Start **Apache** & **MySQL**

### 4️⃣ Create Database

* Go to `http://localhost/phpmyadmin/`
* Create a database named: `chatapp`

### 5️⃣ Import SQL File

* Open `chatapp` database → **Import**
* Select `chatapp.sql` from the project folder → **Go**

### 6️⃣ Run the App

* Visit: `http://localhost/chatapp/`
* Start chatting! 💬

## 🚀 Usage
